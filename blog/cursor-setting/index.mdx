## 使用办法

1. 覆盖保存
    - 下载 <a href="/settings.json" download="settings.json">settings.json</a> 到 `C:\Users\用户名\AppData\Roaming\Cursor\User` 目录下
    - 下载 <a href="/global.code-snippets" download="global.code-snippets">global.code-snippets</a> 到 `C:\Users\用户名\AppData\Roaming\Cursor\User\snippets` 目录下

2. 使用 `zixulu` 进行同步

    ```bash npm2yarn
    npx zixulu sc
    ```
